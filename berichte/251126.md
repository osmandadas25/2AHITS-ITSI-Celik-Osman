## Übung 2.6 (in/out EOF)
```
#include <stdio.h>

int main()
{
    int sum = 0;
    int zahl;

    while (scanf("%d", &zahl) != EOF)
    {
        sum += zahl;
    }

    printf("%d\n", sum);
    return 0;
}

```

## Übung 2.7 (C Hallo Welt)

```sh
echo '#include <stdio.h>' >hello.cpp
echo  >>hello.cpp
echo 'int main()' >>hello.cpp
echo '{'  >>hello.cpp
echo 'printf("Hello World\n");'  >>hello.cpp
echo 'return 0;'  >>hello.cpp 
echo '}'  >>hello.cpp
g++ hello.cpp -o hello
./hello
```
