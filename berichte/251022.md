# Übung 4 Bewegen und Kopieren
## Ausgehend von der, in der vor-vorigen Übungen angelegten, Directory und Filestruktur:

- Nenne neu.txt in umbenannt.txt um.
- Bewege das Verzeichnis first_dir in das Verzeichnis second_dir
- Bewege umbenannt.txt zwei Verzeichnisebenen höher (d.h. nach second_dir)
- Kopiere first_dir inklusive des Inhalts in ein neues Verzeichnis kopie_von_first im Verzeichnis second_dir (Hinweis:   beim Kopieren von Directories mit cp muss die Option -r).
- Lösche das Verzeichnis first_dir.

```sh
mv abcd/first_dir/in_first_dir/neu.txt abcd/first_dir/in_first_dir/umbenannt.txt

mv abcd/first_dir abcd/second_dir/

cd abcd/second_dir/first_dir/in_first_dir/
mv umbenannnt.txt ../../

cp abcd/second_dir/first_dir abcd/second_dir/kopie_von_first -r

rm -r abcd/second_dir/first_dir
```

# 5 Challenge
## Informiere dich über die Optionen der ls, mv und rm Kommandos. Probiere aus und dokumentiere deine Erkenntnisse im Arbeitsbericht.

```
ls -m : Setzt ein Komma zwoschen der Ordnern 
ls -r : Listet in der umgkehrten Reihefolge
ls -S : Beginnt mit der größten Datei

mv -b : Erstell zusätlich ein Backup
mv -n : Überschreibt keine existierende Datei
mv -T : Das Ziel soll keine Verzeichnisbehandlung erfahren 

rm -d : Löscht leere Ordner
rm -r : Löscht ales (kein Undo)
rm -f: ignoriert nicht existierende Dateien

"-v" erklärt was das macht
```
