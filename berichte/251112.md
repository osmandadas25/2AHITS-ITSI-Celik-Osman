### Jedes Programm in Linux haben einen stdin (Standardinput) und einen stdout (Standardoutput). Hierbei kann der stdin die Tastatur ein aber zb. eine Datei. Stdout kann die Konsole sein oder auch eine Datei. < wird für Input und > für Output verwendet.
### Mit cat kann man in eine Datei schreiben oder sie ausgeben. 
```sh
cat >Hallo.txt
```
### Die nächsten Eingaben von der Tastatur (Text) werden in Hallo.txt gespeichert (Um rauszugehen STRG-D oder STRG-C)
```sh
cat <Hallo.txt  
oder
cat Hallo.txt
```
### Liest den Inhalt der Datei in der Konsole vor
### Man kann auch echo verwenden um in eine Datei zu schreiben
```sh
echo (text) >Hallo.txt
```
### Überschreibt die Daten in der Datei
```sh
echo (text) >>Hallo.txt
```
### Schreibt den Text in eine neue Zeile (Der ursprüngliche Inhalt bleibt uneversehrt)

### Hashtags werden als Kommentare verwendet in Linux. Quotes werden mit Hochkomma geschrieben
```sh
mkdir 'Weltmeisterschaft Qatar'
```

## Übung Kopie mit cat
#### Erstelle mit Hilfe von “cat” eine Kopie der Datei test.txt in der neuen Datei test2.txt. test2.txt soll den gleichen Inhalt enthalten wie test.txt. Prüfe ob test2.txt wirklich den gewünschten Inhalt hat.

```sh

```

## Übung Textdatei erstellen
#### Mit Hilfe von mehreren echo Befehlen können einfache Textdateien erstellt werden. Erzeuge nur unter Verwendung von echo Befehlen die Datei made_by_echoing.txt mit Inhalt:

#### =====================
#### =    HTL BRAUNAU    =
#### =====================
#### = 2AHITS Gruppe n   =
#### = x Schülerinnen    =
#### = y Schüler         =
#### =====================
#### Setze für n, x und y die richtigen Werte ein.




```sh
echo ===================== >made_by_echoing.txt
echo ='    HTL BRAUNAU    '= >>made_by_echoing.txt
echo ===================== >>made_by_echoing.txt
echo =' 2AHITS Gruppe 1   '= >>made_by_echoing.txt
echo =' 0 Schülerinnen    '= >>made_by_echoing.txt
echo =' 12 Schüler        '= >>made_by_echoing.txt
echo ===================== >>made_by_echoing.txt
```
```sh
# Ohne die Hochkomma wurde es so ausgegeben: 
=====================
= HTL BRAUNAU =
=====================
= 2AHITS Gruppe 1 =
= 0 Schülerinnen =
= 12 Schüler =
=====================
```
