### In Linux ist das Arbeitsverzeichnis (Working Directory) das aktuelle Verzeichnis auf der Kommandozeile, in dem Befehle ausgeführt und Dateien bearbeitet werden

### Der Unterschied besteht darin, dass ein absoluter Pfad den vollständigen Speicherort einer Datei oder eines Verzeichnisses vom Stammverzeichnis (/) des Dateisystems angibt, während ein relativer Pfad den Speicherort in Bezug auf das aktuelle Verzeichnis beschreibt und nicht mit der Wurzel beginnt

Mithilfe dieses Befehls kann man was ausgeben sowie printf.
```sh
$ echo
```
Dieser Befehl zeigt den Inhalt eines Verzeichnisses.
```sh
ls 
```
Dieser Befehl wird verwendet, um das aktuelle Arbeitsverzeichnis im Terminal zu wechseln.
```sh
cd 
```
Dieser Befehl zeigt den vollständigen Pfad des aktuellen Verzeichnisses an, in dem Sie sich gerade im Terminal befinden.
```sh
pwd
```

```
/home/runner/workspace
```

Dieser Befehl zeigt dass eine Datei vorhanden ist.
```sh
ls -l
```

```
total 8
drwxr-xr-x 1 runner runner  90 Oct  8 07:27 berichte
-rw-r--r-- 1 runner runner 102 Sep 24 06:59 README.md
drwxr-xr-x 1 runner runner   0 Oct  8 07:15 test
```

Diesen Befehl nutzt man um Unterverzeichnisse zu erstellen.
```sh
mkdir
```
Mit Hilfe dieses Befehls kann man Dateien erstellen.
```sh
touch
```
Durch diese Aktion (remove) wird die danach eingegebene Datei gelöscht. (Es gibt kein Undo)
```sh
rm
```
Dieser Befehl löscht (remove directory) ein bestimmtes und leeres Unterverzeichnis.
```sh
rmdir
```
Dieser Befehl löscht (remove directory) ein bestimmtes Unterverzeichnis.
```sh
rm -r
```
Der Befehl löscht alles was vom Root-Verzeichnis ausgehend.
```sh
rm -r /
```
Mit dem Befehl kann man in das obere Verzeichnis wechseln.
```sh
cd ..
```
Man Muss nicht immer das Arbeitsverzeichnis wechseln um eine Datei in einem Unterverzeichnis zu erstellen/löschen.
```sh
touch test/neu.txt
rm test/neu.txt
```
### In Linux steht / für das Root-Verzeichnis und ~ für das Home-Verzeichnis des aktuellen Benutzers.
Dieser Befehl wechelt das Arbeitsverzeichnis zum Wurzelverzeichnis/Homeverzeichnis.
```sh
cd /
cd ~
```
Dieser Befehl zeigt den Inhalt der Wurzelverzeichnisses/Homeverzeichnisses.
```sh
ls /
ls ~
```
### Kurze Zusamenfassung: 
- .  : Das aktuelle Verzeichnis
- .. : Das obere Verzeichnis
- /  : WurzelVerzeichnis (Root).
- ~  : Homeverzeichnis des aktuellen Benutzers
