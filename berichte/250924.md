  # Grundlagen der Linux Shell

  Shell = textorientierte Schnitttelle zu einem Betriebsystem (Kernel)

  Anzeige in der Konsole: Prompt
  Eingabe nach dem Prompt

  Beispiel : Der Befehl `date`

  ```sh
  $ date
  ```

  Ausgabe:
  
  ```
  Wed Sep 24 07:11:27 AM UTC 2025
  ```

  Befehle können Optionen haben, diese werden mit `-` angegeben.

  z.b.

  ```sh
  $ date -I
  ```
  Ausgabe:
  ```
  2025-09-24
  ```

  Optionen aus einem Buchstaben = shorthand

  lonhhand mit 2x `--`;

  ```sh
  $ date --iso8601
  ```

  Mehr Informationen zu einem Befehl:

  

  ```sh
  $ date --help
  ```

  - man pages

  ```sh
  $ man date
  ```
  nicht in Replit aber man kann mit dem Text `man date` im Internet suchen.

  - KI
  - häufig gibt es die Option `--help`
